#TestNG-Demo
This is TestNG related repository where we have covered allmost all the features in TestNG.

Features Covered:
1. This is Maven Project
2. TestNG dependencies added
3. package created "com.qa.testcases1"
4. Project converted to TestNG: Rightclick on Project-> TestNG-> ConvertTo TestNG
5. Controlling Testcase execution from TestNG.xml files
6. Include and Exclude mechanism
7. Groups
8. Parameterization in TestNG.xml and 
9. DataProvider
10. Listeners in TestNG as interface
